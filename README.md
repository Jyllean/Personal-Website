# Personal Web Profile - Engr. Ma. Angelica Alzaga

A modern, responsive personal web profile built with HTML, CSS, and JavaScript. Created as part of the CMSC 207 Mini-Project, this website showcases my journey from Civil Engineering to Computer Science at the University of the Philippines Open University.

## 🌟 Features

- Semantic tags (`<header>`, `<section>`, `<footer>`, etc.)
- Mobile-first approach with CSS media queries for all screen sizes
- Fun fact button with dynamic content display
- CSS transitions and keyframe animations throughout

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling, animations, and responsive design
  - CSS Variables for theming
  - Flexbox and Grid for layout
  - Media queries for responsiveness
  - Keyframe animations
- **JavaScript (Vanilla)** - Interactive functionality
  - DOM manipulation
  - Event listeners
  - Scroll reveal animations
  - Intersection Observer API
- **Vercel** - Hosting and Deployment

## ✨ JavaScript Functionality

### Fun Fact Feature
- Displays random fun facts from an array
- Prevents showing the same fact twice in a row
- Smooth animation when displaying new facts
- Button bounce effect on click

### Scroll Animations
- Sections fade in as user scrolls
- Hobby cards stagger animation on scroll
- Smooth scroll for navigation links
- Active navigation state based on scroll position
