## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/personal-web-profile.git
```

2. Navigate to the project directory:
```bash
cd personal-web-profile
```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local server (Live Server extension in VS Code recommended)

### Customization

This website is personalized for Engr. Ma. Angelica Alzaga. To further customize:

1. **Add Your Photos**:
   - **Profile photo**: Replace `profile/profile.jpg` with your photo (square, ~400x400px)
   - **Cover photo**: Replace `profile/cover.jpg` with your background image (16:9 ratio, ~1600x900px)
   - **Travel photos**: Add 4 photos to `images/` folder named `travel1.jpg` through `travel4.jpg`

2. **Modify Fun Facts** in `script.js`:
   - Edit the `funFacts` array with additional personal facts

3. **Adjust Colors** in `styles.css`:
   - Modify CSS variables in the `:root` selector
   - Change color scheme to match your preference
